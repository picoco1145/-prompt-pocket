# Prompt Extension 설계 문서

> 작성일: 2025-01-27

## 개요

프롬프트를 저장하고 AI 챗봇 사이트에서 빠르게 사용할 수 있는 Chrome 확장 프로그램

## 타겟 사이트

- ChatGPT (chat.openai.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)
- Notebook LM (notebooklm.google.com)

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 프롬프트 저장 | 태그로 체계적 관리 |
| 변수 지원 | `{{변수명}}` 형식, 선택적 사용 |
| 빠른 사용 | 팝업에서 선택 → 복사 |
| 사용 통계 | 자주 쓰는 순, 최근 사용 순 정렬 |
| 동기화 | Chrome 계정 동기화 + Google Drive 백업 |

## 프로젝트 구조

```
prompt-extension/
├── manifest.json          # 확장 프로그램 설정 (Manifest V3)
├── popup/                 # 팝업 UI (프롬프트 선택/사용)
│   ├── popup.html
│   ├── popup.css
│   └── popup.js
├── options/               # 관리 페이지 (프롬프트 CRUD)
│   ├── options.html
│   ├── options.css
│   └── options.js
├── scripts/
│   └── clipboard.js       # 클립보드 복사 유틸 (예정)
├── lib/
│   ├── storage.js         # Chrome Storage 관리
│   └── drive.js           # Google Drive 백업/복원
└── assets/
    └── icons/             # 확장 프로그램 아이콘
```

## 기술 구현

### Chrome Storage 전략

```javascript
// chrome.storage.sync (자동 기기 간 동기화, 100KB 제한)
{
  tags: [...],
  settings: { theme: 'light', defaultSort: 'recent' }
}

// chrome.storage.local (로컬 전용, 5MB 제한)
{
  prompts: [...],
  lastSyncedAt: 1706400000
}
```

### Google Drive 연동 (MVP)

- OAuth 2.0으로 Google 로그인
- Drive API의 appDataFolder 사용 (앱 전용 숨김 폴더)
- 수동 백업: prompts.json 파일로 Drive에 저장
- 수동 복원: Drive에서 prompts.json 불러오기
- 추후 자동 동기화로 확장 가능

## 개발 접근 방식

**MVP 우선 개발:**
1. 핵심 기능 먼저 완성 (저장, 태그, 사용)
2. Chrome Storage Sync로 기기 간 동기화
3. Google Drive는 수동 백업/복원으로 구현
4. 점진적으로 기능 확장
