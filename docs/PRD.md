# Prompt Pocket - PRD

> 버전: 1.0
> 작성일: 2025-01-27

## 1. 개요

**Prompt Pocket**은 자주 사용하는 프롬프트를 저장하고, AI 챗봇 사이트에서 빠르게 불러와 사용할 수 있는 Chrome 확장 프로그램이다.

- **핵심 가치**: 반복되는 프롬프트 입력 시간 절약, 체계적인 프롬프트 관리
- **타겟 플랫폼**: ChatGPT, Claude, Gemini, Notebook LM
- **개발 방식**: MVP 우선 개발 후 점진적 확장

## 2. 문제 정의

**현재 문제점:**
- AI 챗봇 사용 시 매번 같은 프롬프트를 수동으로 입력
- 좋은 프롬프트를 메모장이나 노션에 따로 저장 → 복사/붙여넣기 번거로움
- 프롬프트가 많아지면 찾기 어려움
- 프롬프트에 변수(예: 언어, 주제)가 있으면 매번 수정 필요

**해결 방안:**
- 브라우저 확장 프로그램으로 프롬프트를 바로 접근 가능하게
- 태그로 체계적 분류
- 변수 기능으로 재사용성 극대화

## 3. 목표 사용자

**주 사용자:** 본인 (개인 프로젝트)

**사용자 특성:**
- AI 챗봇(ChatGPT, Claude, Gemini)을 일상적으로 사용
- Notebook LM으로 문서 분석 작업 수행
- 반복적으로 사용하는 프롬프트 패턴이 있음
- 효율적인 워크플로우를 선호

## 4. 핵심 기능

### 4.1 프롬프트 저장 및 관리
| 기능 | 설명 |
|------|------|
| 프롬프트 CRUD | 생성, 조회, 수정, 삭제 |
| 태그 시스템 | 복수 태그 부착, 색상 지정 가능 |
| 검색 | 제목, 내용, 태그 통합 검색 |

### 4.2 변수 기능
- **MVP 범위에서 제외 (보류)**
- 추후 `{{변수명}}` 기반 치환 기능을 재검토

### 4.3 프롬프트 사용
| 방식 | 설명 |
|------|------|
| 클립보드 복사 | 프롬프트를 클립보드에 복사 |

### 4.4 사용 통계
- 프롬프트별 사용 횟수 기록
- 최근 사용 시간 기록
- 자주 쓰는 순 / 최근 사용 순 정렬

### 4.5 데이터 저장 및 동기화
| 데이터 | 저장 위치 | 동기화 |
|--------|----------|--------|
| 태그, 설정 | Chrome Sync | 자동 (구글 계정) |
| 프롬프트 | Local Storage | 수동 (Google Drive 백업) |

> **참고**: 프롬프트는 용량 제한(100KB)으로 인해 로컬 저장 후 Google Drive로 수동 백업/복원

## 5. 사용자 시나리오

### 시나리오 1: 프롬프트 저장
1. ChatGPT에서 유용한 프롬프트 발견
2. Prompt Pocket 아이콘 클릭 → [빠른 추가]
3. 제목, 태그 입력 후 저장

### 시나리오 2: 프롬프트 사용 (변수 없음)
1. Claude 사이트 접속
2. Prompt Pocket 아이콘 클릭
3. "코드 리뷰 요청" 프롬프트 선택
4. [복사하기] 클릭 → 입력창에 붙여넣기

### 시나리오 3: 프롬프트 사용 (변수 없음)
1. Gemini 사이트 접속
2. Prompt Pocket 아이콘 클릭
3. "번역 요청" 프롬프트 선택
4. [복사하기] 클릭 → 입력창에 붙여넣기

### 시나리오 4: 프롬프트 관리
1. 팝업에서 [관리] 클릭 → 관리 페이지 열림
2. 여러 프롬프트 선택 → 태그 추가/삭제
4. Google Drive로 백업

## 6. 기술 요구사항

### 6.1 Chrome Extension
- **Manifest Version**: V3 (최신 표준)
- **권한**: storage, contextMenus, clipboardWrite
- **지원 사이트**:
  - chat.openai.com
  - claude.ai
  - gemini.google.com
  - notebooklm.google.com

### 6.2 데이터 저장
```
chrome.storage.sync (100KB 제한)
├── tags: 태그 목록
└── settings: 설정값

chrome.storage.local (5MB 제한)
├── prompts: 프롬프트 데이터
└── lastSyncedAt: 마지막 동기화 시점
```

### 6.3 Google Drive 연동
- OAuth 2.0 인증
- Drive API - appDataFolder 사용
- 백업 파일 형식: JSON

### 6.4 UI/UX
- 심플/미니멀 디자인
- 라이트 테마
- 반응형 팝업 (400px 너비)
